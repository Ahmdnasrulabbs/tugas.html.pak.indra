Halaman Web Praktikum 3 - HTML

Ini adalah halaman web yang dibuat untuk memenuhi tugas Praktikum 3. Halaman ini menggunakan berbagai tag HTML.

Pengertian HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur halaman web.

Tujuan HTML

HTML digunakan untuk membuat dokumen yang dapat ditampilkan di browser web. HTML juga memungkinkan pengembang untuk menambahkan berbagai elemen seperti teks, gambar, tautan, dan form.

Tag HTML yang Digunakan

Berikut adalah beberapa tag HTML yang digunakan dalam halaman ini:

  • <h1> - Heading utama
  • <h2> - Sub-heading
  • <p> - Paragraf
  • <a> - Tautan
  • <img> - Gambar
  • <ul> - Daftar tidak berurutan
  • <ol> - Daftar berurutan
  • <table> - Tabel

Contoh Daftar Berurutan

  1. HTML adalah dasar dari pembuatan halaman web.
  2. CSS digunakan untuk styling dan layout.
  3. JavaScript digunakan untuk interaktivitas.
